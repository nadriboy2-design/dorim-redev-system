## 🔒 규칙 2 — 법적 하드락 (Hard Lock)

다음 3가지는 시스템이 **자동으로 차단**해야 하며 사용자가 우회할 수 없습니다.

### 하드락 A — 용적률 상한 초과 차단
```
IF 사용자_입력_용적률 > 700%:
    RAISE ComplianceError("1차 역세권 법적 상한 700% 초과. 서울시 역세권 운영기준 제○조 위반.")
    BLOCK 해당_문서_생성
```

### 하드락 B — 시공사 선정 시기 잠금
```
IF 현재_사업_단계 != "조합설립완료" AND 요청_문서_유형 == "시공사입찰공고":
    RAISE ComplianceError("서울시 도시 및 주거환경정비 조례에 의거, 시공사 선정은 조합설립인가 이후에만 가능합니다.")
    BLOCK API_CALL
```

### 하드락 C — 공사비 검증 없는 분양공고 차단
```
IF 공사비_검증_결과서 == NULL AND 요청_문서_유형 == "분양공고":
    RAISE ComplianceError("국토교통부 고시에 따른 정비사업 공사비 검증 결과서가 첨부되지 않았습니다.")
    BLOCK API_CALL
```
